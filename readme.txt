SimpanKira integration for WooCommerce.

== Description ==

SimpanKira offers a cloud accounting solutions for small business. We include accounting, invoicing and billing, payment tracking, project accounting, inventory, web POS and HR & Payroll. This plugin is to synchronize WooCommerce transactions to your SimpanKira cloud accounting.

Visit https://simpankira.com for more information.

== Features ==

- Automatically synchronises total daily sales value from the completed Order in WooCommerce to SimpanKira Cloud Accounting on daily basis
- Allow manual synchronises total daily sales value from the completed Order in WooCommerce to SimpanKira Cloud Accounting based on specified date

== Installation ==

1. Log in to your WordPress admin.
2. Search plugins "SimpanKira for WooCommerce" and click Install Now.
3. Activate the plugin.
4. Navigate to "SimpanKira" menu in the sidebar to access the plugin settings page.
5. Follow the instructions and update the plugin settings.
6. Done!
